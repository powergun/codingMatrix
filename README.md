# Python Linear Algebra and Applications

Some contents are inspired by coding the matrix book.

## Project Structure

`src/pycodingmatrix`

implementing the examples in the book, coding the matrix, and making
these functions available for the jupyter notebooks.

some of these functions have unit tests in the `tests` directory;

`src/symbolic`

documenting how to write linear algebra related math symbols properly
in jupyter notebook;

see also: `pyMath/symbolic` for a complete cheatsheet
